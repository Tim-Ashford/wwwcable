ADSL/PSTN CABLE TESTING


Physical telephone lines in Australia require two essential types of test to disclose faults in the copper or aluminium, balanced, twisted cable pairs.

The first test type - DC (Direct Current) testing ensures that no unwanted DC voltages and / or resistive losses exist within the wire pairs. This is achieved with the Telstra (compulsory) F.A.S.T. test or its equivalent.

The second test type - AC (Alternating Current) testing is to ensure that the same cable pair is capable of carrying Speech, Tone signalling, DSL data tones, Pulse signalling, CLI and metering signals when required. All of these signals are used over a standard modern PSTN service in Australia but astoundingly this test is rarely, if ever performed, leaving several fault conditions (Attenuation and Impedance faults) undetected on the allegedly "Tested" lines.
